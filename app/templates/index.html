DaF Sprachdiagnostik v2

Deutsch-Einstufungstest

Finden Sie heraus, welche Grammatikmodule Sie gezielt üben sollten – in ca. 30–45 Minuten.

Test starten

Wenn Sie einen persönlichen Zugangscode erhalten haben, geben Sie ihn bitte hier ein. Ohne Code können Sie den Test als Gast starten.

Der Test umfasst 6 Module:

📝 M1: Lückentext (Grammatik) 📖 M2: Leseverstehen 🎧 M3: Hörverstehen 🗣️ M4: Vorlesen 💬 M5: Freies Sprechen ✍️ M6: Schreiben
Die Auswertung basiert auf Hueber Schritte plus Neu (Bände 1–4). Am Ende erhalten Sie ein persönliches Skill-Profil und gezielte Kursempfehlungen.
